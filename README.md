# ふくざうるすチャット

福井県を案内してくれる、ゆるい恐竜チャットボットです。
GitHub Pages にそのまま置いて動きます。
現代Web開発なのにビルド不要です。文明が少しだけ正常だった頃の空気。

---

## 🦖 デモ

「ふくざうるす」が福井県の観光地やお店を案内します。

特徴:

* 恐竜キャラクター
* 日本語 + English の2言語返答
* OpenAI系チャットAPI利用
* 単一HTMLで動作
* GitHub Pages対応
* スマホ対応

## 🌐 GitHub Pages で公開

Repository の:

```text
Settings
→ Pages
→ Deploy from branch
→ main / root
```

を選択。

数十秒で公開されます。
